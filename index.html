How to make website in this application

For content editors

  • Edit text directly in this editor. (Try on this) editedit d eee dd
  • Drag and drop images. Image files ending @2x and @3x are optimized
  • Duplicate or remove blocks (click to this to duplicate)

For source editors

  • Switch to View source
  • Edit any html or css file
  • Add data-editor="editable", data-editor="image", data-editor="duplicable", data-editor="removable"
